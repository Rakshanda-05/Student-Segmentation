# 📊 Student Segmentation Analysis using Machine Learning

A Machine Learning project that analyzes student academic performance and groups students into meaningful clusters using unsupervised learning techniques. The project helps educational institutions understand learning patterns and support data-driven decision making.

---

## 🚀 Features

- Data preprocessing
- Missing value handling
- Outlier detection
- Feature engineering
- Exploratory Data Analysis (EDA)
- K-Means Clustering
- Hierarchical Clustering
- Interactive Power BI Dashboard
